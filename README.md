# IO2020
To start do steps listed below:
  1. Clone the repo.
  2. Download YOLOv3 weights .weights file: https://pjreddie.com/media/files/yolov3.weights
  3. ```pip install -r requirements.txt```
  5. Make sure your folder looks like that:
  ```
      IO2020
      |   .gitignore
      |   backend.py
      |   coco.names
      |   gui.py
      |   output.txt
      |   README.md
      |   requirements.txt
      |   yolov3.cfg
      |   yolov3.weights          
      +---venv
      ```
